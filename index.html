What is Recon?

+

$Recon is a tool or service that evaluates tokens on the Solana ecosystem for security risks.

What does Recon do?

+

It uses advanced algorithmic tools to analyze tokens and identify potential security vulnerabilities.

What is the purpose of Recon?

+

To provide the best security solutions for the Solana ecosystem by identifying and mitigating risks associated with tokens.

Is Recon secure to use?

+

Security is our top priority. Recon uses non-custodial technology, meaning we never have access to your private keys or funds.

Yes, Recon is fully responsive and works on mobile devices. However, for the best trading experience with advanced features, we recommend using a desktop interface.
